## API Details

**Service:** wttr.in (free weather API)  
**Endpoint:** `https://wttr.in/{city}?format=j1`  
**Auth:** None (no API key needed)  
**Coverage:** 15,000+ cities worldwide

## Troubleshooting

| Issue | Fix |
|-------|-----|
| ModuleNotFoundError | `pip install requests` |
| City not found | Use English name or "City, Country" |
| Timeout | Check internet connection |
| Port 5000 busy | Change port in weather_flask_app.py |

## Example Uses

**Get weather for any city:**
